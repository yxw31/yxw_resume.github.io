## TL;DR

LADRC 的三个核心参数：

| 参数 | 含义 | 典型范围 |
|---|---|---|
| `b0` | 控制增益（系统关键先验） | 经验起点：`b0 ≈ 1 / J_eq` |
| `ω_o` | 扩张状态观测器带宽 | 控制带宽 ω_c 的 **3~5 倍** |
| `ω_c` | 控制器带宽 | 由实际性能需求定 |

## 为什么 ω_o = 3~5 × ω_c

ESO 必须比控制器**响应更快**才能"先一步"看到扰动，否则估出来的扰动滞后，反而把闭环搞坏。
但 ω_o 太大会放大测量噪声 —— 经验上 3-5 倍是甜点。

## b0 怎么估

LADRC 对 `b0` 容错性其实不差（这正是它"自抗扰"的好处），但**起点要在正确的数量级**：

```
b0 ≈ Kt / J_eq
其中:
  Kt    电机转矩常数 (Nm/A)
  J_eq  折算到电机轴的等效转动惯量
```

实测 ISG 电机调参：

```matlab
% 设计起点
J_eq = 0.012;        % kg·m²
Kt   = 1.85;         % Nm/A
b0   = Kt / J_eq;    % ≈ 154

omega_c = 80;        % rad/s, 转速环
omega_o = 4 * omega_c;  % = 320

% 控制器
Kp = omega_c;
% ESO 增益
beta1 = 2 * omega_o;
beta2 = omega_o^2;
beta3 = ...
```

## 调参流程

1. 估算 `b0`，先固定不动
2. 拉 `ω_c` 到稳定但不振荡的最大值
3. 调 `ω_o`，从 3·ω_c 起，逐步增大到出现噪声敏感为止
4. 微调 `b0`（±30% 范围）观察稳态误差是否减小
5. 跑变负载工况验证

## 与 PI 对比

| 工况 | PI | LADRC |
|---|---|---|
| 阶跃响应 | 好 | 好 |
| 变负载 (10→100%) | 转速跌 ~5% | 转速跌 < 1% |
| 模型不确定 | 需要重新整定 | 鲁棒性高 |
| 参数数量 | 2 (Kp, Ki) | 3 (b0, ω_o, ω_c) |

LADRC 优势主要在 **2&3** 两栏。代价是需要**多估计一个状态**，CPU 占用稍高。

## 工程提醒

- **离散化用 Tustin** 比 Forward Euler 在高带宽下稳定得多
- **限幅** 一定要给在控制量和 ESO 估计上，否则崩起来很快
- **变结构时**（比如切到弱磁）b0 应该跟着变，不然抗扰能力降级

ISG 电机变负载 30→100% 阶跃实测：
- 仅 PI：转速下跌 ~85 rpm，恢复时间 220 ms
- LADRC：转速下跌 ~22 rpm，恢复时间 95 ms

提升明显，但 ESO 在极低速（< 50 rpm）下会有抖动，**低速段切回 PI** 是工程上的妥协。
